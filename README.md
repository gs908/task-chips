# Task Chips

A task management system based on Directed Acyclic Graph (DAG) with AI-powered task expansion capabilities.

## 🎯 Core Features

- **DAG-Based Task Management**: Maintain requirement nodes as a directed acyclic graph
- **Tree Structure Visualization**: Display tasks in an intuitive tree hierarchy
- **Progressive Refinement**: Break down tasks layer by layer
- **AI-Powered Expansion**: Expand individual tasks using AI assistance

## 📋 Project Overview

Task Chips enables you to:
1. Create complex task structures with dependencies
2. Visualize tasks in tree format
3. Refine tasks progressively from high-level to detailed
4. Use AI to expand and suggest subtasks for any node

## 🚀 Getting Started

### Prerequisites

- Node.js 18+ 
- npm, yarn, or pnpm
- Git

### Installation

```bash
# Clone the repository
git clone https://github.com/gs908/task-chips.git
cd task-chips

# Install dependencies
npm install

# Run development server
npm run dev
```
